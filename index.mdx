Pika Skills are self-contained modules you install into AI coding agents like Claude Code and OpenClaw. Once installed, the agent automatically detects the skill and knows how to use it — no manual configuration required.

Each skill extends your agent with capabilities backed by the [Pika Developer API](https://www.pika.me/dev/), such as joining video meetings as a real-time AI avatar, cloning voices, and more.

## How skills work

<Steps>
  <Step title="Get a Pika Developer Key">
    Sign up at [pika.me/dev](https://www.pika.me/dev/) to get a Developer Key starting with `dk_`. This key authenticates all API calls made by your skills.
  </Step>
  <Step title="Set your environment variable">
    Export your key so the skill scripts can authenticate:

    ```bash
    export PIKA_DEV_KEY="dk_your-key-here"
    ```
  </Step>
  <Step title="Install a skill">
    Point your AI agent at the skill directory and ask it to install:

    ```
    install /path/to/Pika-Skills/pikastream-video-meeting/
    ```

    The agent reads `SKILL.md` and handles everything automatically.
  </Step>
  <Step title="Use it naturally">
    Interact with your agent as you normally would. For example, drop a Google Meet link and the agent will automatically activate the video meeting skill — checking your balance, setting up your avatar, and joining the call.
  </Step>
</Steps>
